# Local Deployment Instructions

## Setup
1. Copy .env.example to .env
2. Update .env with your API keys:
   - OPENAI_API_KEY=your-openai-key-here
   - SUPABASE_URL=your-supabase-url
   - SUPABASE_SERVICE_ROLE_KEY=your-supabase-key

## Install Dependencies
pip install -r requirements.txt

## Run Application
python -m src.main

## Access Application
- Main Interface: http://localhost:8001
- Flow: /auth → /welcome → /persona-select → /chat

## Required Environment Variables
- OPENAI_API_KEY: Required for AI responses
- SUPABASE_URL: Database connection
- SUPABASE_SERVICE_ROLE_KEY: Database access
- SUPABASE_ANON_KEY: Frontend auth (optional)

## Services Used
- enhanced_persona_service: Main AI persona interactions
- enhanced_chat API: /api/chat/start, /api/chat/send, /api/chat/personas
- Static file serving: auth, welcome, persona-select, chat pages
